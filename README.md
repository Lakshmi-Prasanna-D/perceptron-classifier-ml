# Perceptron Classifier (Machine Learning Lab)

This repository contains Python implementations of the Perceptron algorithm.

Programs included:
1. Linear Perceptron classifier
2. Quadratic classifier using g(x) = a + bx + cx^2

Features:
- User input patterns
- Automatic feature augmentation
- Weight updates during misclassification
- Displays g(x) values used for classification
- Final converged weight vector

Language: Python
